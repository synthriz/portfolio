# Meu portfolio <3
Desenvolvido com React + TypeScript + Vite + Tailwind CSS, com suporte a PT/EN, modo claro/escuro e uma página de CV pronta para leitura e impressão.

### Stack

- React 18

- TypeScript

- Vite

- Tailwind CSS 4

- React Router

- React Icons

### Rotas

- / → portfólio

- /cv → currículo
